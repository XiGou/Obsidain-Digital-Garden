| **概念**           | **润色后的表达**                                                  |
| ---------------- | ----------------------------------------------------------- |
| **NPD（自恋型人格障碍）** | 你不能表现出自恋型人格障碍（NPD）的特征，即不能只关注自己的感受，而应更多地关注我的需求并给予我支持。        |
| **原生家庭**         | 我之所以这么坏，主要源于原生家庭的创伤经历。                                      |
| **讨好型人格**        | 我倾向于接受你的善意而不予回报，因为我不愿意成为讨好型人格。                              |
| **反驳性人格**        | 我希望我的观点被无条件接受，如果你提出不同意见，说明你具有反驳性人格。                         |

| **拒绝标签**         | 我因在传统社会价值体系中被严重低估而被贴上众多负面标签，因此选择拒绝社会标签，通过创造属于自己的价值体系来保持优越感。 |
