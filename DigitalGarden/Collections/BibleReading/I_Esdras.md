本篇主要涉及几个人物：  

| 人物       | 身份                           |
| ---------- |:------------------------------ |
| 居鲁士王   | 是波斯国的一位帝王             |
| 亚达薛西王 | 波斯帝国居鲁士王之后的一位帝王 |
| 大流士王   | 亚达薛西之后的一位帝王         |
| 以斯拉     | 以色列的一位祭司               |
| 尼布甲尼撒 | 巴比伦王                       |

本篇开篇故事是居鲁士王开始信奉耶和华，并且允许以色列人重修被烧毁的耶路撒冷圣城和圣殿。而在后续的亚达薛西王时期，耶路撒冷周边各民族以及波斯官僚联名上书给王说以色列人自古以来就擅长违抗王命、热爱造反，因此亚达薛西王下令禁止以色列人继续修建圣殿。在大流士王时期，以色列人又一次被允许继续修建圣殿。

以斯拉则主要活动在亚达薛西时代，他是祭司长亚伦的后代，他收到亚达薛西王的任命，负责按照耶和华的法律执行耶路撒冷地区的审判工作，这大约相当于当今的特别行政区和自治区首长的地位。

这其中有少量通过第一人称叙事的文字，主要讲述了以色列人和外族通婚的事件，当时很多的以色列人都娶了外族的女子，以斯拉带领以色列人在耶和华面前忏悔，并且下令三天之内赶走全部外族女子，否则就抄家。以色列人只能服从，将外族的妻子全部送走了。

本篇的重要信息就是以色列人目前被波斯统治，但是波斯人又无法直接统治以色列，所以实行了代理统治策略，以斯拉就是被波斯人选出来的走狗，负责帮助波斯人统治以色列人。
