本篇主要涉及人物：

| 人物       | 身份                           |
| ---------- |:------------------------------ |
| 尼希米     | 国王身边的御酒官               |
| 亚达薛西王 | 波斯帝国居鲁士王之后的一位帝王 |
| 以斯拉     | 以色列的一位祭司               |

这篇经文的时间线和以斯拉是重叠的，两者同时发生，不过本篇通过尼西米的视角描述了重建耶路撒冷的过程。

尼西米作为亚达薛西王的御酒官，他在国王面前愁眉苦脸，于是国王就询问他原因，他向国王诉说了自己的族人遭受的苦难以及圣城遭毁灭的现状，于是国王答应了他，赐予他诏书并且护送他前往耶路撒冷。在尼西米的组织之下，以色列人团结起来，共同重建耶路撒冷，过程中受到许多阻挠，以色列人甚至需要派一半的人保卫安全，另一半人修建城墙来保证工程不受敌人破坏。

在城墙修建完成之后，尼西米立刻组织大家聚集起来，在抄经士以斯拉的带领下，深刻学习摩西法典，他们连续七天阅读摩西法典，并且在第八天举行大会庆祝，当然免不了反复强调神迹、忏悔对神的背叛、表忠心，确定献祭的频次和数量。

后续则是纠正了一些以色列人的背叛行为，比如不献“十一税”，娶了外族女子等，在尼西米的祷告和劝说下，这些人纠正了自己的行为。
