
[Tabby - a terminal for a more modern age](https://tabby.sh/)  
功能很好，但是 UI 界面不够美观，没有经过良好设计。

[GitHub - kingToolbox/WindTerm: A professional cross-platform SSH/Sftp/Shell/Telnet/Serial terminal.](https://github.com/kingToolbox/WindTerm)  
免费软件，但是不开源，功能非常齐全，界面复杂简陋但是实用。

[Termius - SSH platform for Mobile and Desktop](https://termius.com/)  
商业软件，但是个人使用基本没有太多限制，各方面都无可挑剔，目前在用。

[Warp: Own the command line](https://www.warp.dev/)  
号称跨平台 + AI，但是 windows 平台一直在 waitlist，有些难绷，由于我需要保证跨平台体验一致，暂未采用。

[Wave Terminal — Upgrade Your Command Line](https://www.waveterm.dev/)  
真跨平台 + AI + 开源，UI 有点花里胡哨，内部有 CPU 负载图表， 还有树形文件目录，带有一个 AI 对话框，甚至内嵌网页浏览器，也不知道有啥用；速度快是其最令人记忆深刻的点，速度快的有些逆天。  
它的设计逻辑是一个本地 Terminal，比如在 Windows 平台，要先进入 powershell 然后使用 ssh 命令连接到远程机器，我的评价是不如 Windows Terminal.
