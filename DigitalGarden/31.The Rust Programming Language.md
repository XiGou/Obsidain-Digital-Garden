# 开始学习 Rust
  
## ownership
堆上的浅拷贝都会导致 ownership 转移，也就是 move 语义， 离开变量的 scope 之后会触发 drop 操作释放内存。操作没有 ownership 的变量，编译时将会检测出来并停止编译。
## reference
可以访问变量但是不转移 ownership， 默认 immutable。mutable reference 只能有一个，这样破坏了下述第一个 race condition，所以编译时即可保证没有 race 产生。
A _data race_ is similar to a race condition and happens when these three behaviors occur:
- Two or more pointers access the same data at the same time.
- At least one of the pointers is being used to write to the data.
- There’s no mechanism being used to synchronize access to the data.
